   Addr - file name 			- Info
--------------------------------------------------------   
0x00000 - bin/0x00000.bin		- Program codes IRAM/RAM
0x0A000 - webbin/WEBFiles.bin	- WebFileSystem
0x40000 - bin/0x40000.bin		- Program codes Cache Flash
0x79000 - bin/clear_eep.bin		- Clear settings EEP area
0x7C000 - bin/esp_init_data_default.bin - RF SDK options
0x7E000 - bin/blank.bin			- Default SDK WiFi config

bin/rapid_loader.bin - loader QSPI CLK x2 (80MHz)
bin/rapid_loader_m40.bin - loader QSPI CLK x1 (40MHz)
